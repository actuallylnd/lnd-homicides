## INSTALATION

You need items then paste it into ox_inventory/data/items.lua

```
	['orderphone'] = {
		label = 'Stolen phone',
		weight = 1,
		stack = false,
	},


	['victimsdrugs'] = {
		label = 'Weird pills',
		weight = 1,
		stack = true,
	},
```

Put images into ox_inventory/web/images dont change name of pictures
